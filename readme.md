## Key Features

* Create a group with name
  - Add Users to Team 1 or Team 2
  - Remove Users from a Team
* Delete Group
* Delete All Groups if have more than 1

## How To Use

```bash
# Install dependencies
$ npm install

# Run the app
$ npx expo start
```
